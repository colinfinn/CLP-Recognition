## python 项目注释规范
该文件注释段写在文件头部，具体函数的注释
写在函数内部，包括了参数返回值等
``` python
# -*- coding: utf-8 -*-
# @Time    : 日期
# @Author  : 作者
# @Sid     : 学号
# @File    : 当前文件名
# @Description : 该程序的功能
```
函数声明如下
```python
def fun(a, b):
    """
    该函数的描述
    :param a: 参数说明
    :param b: 参数说明
    """
    函数的具体实现
```
